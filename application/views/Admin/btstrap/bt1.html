Bootstrap

Bootstrap是一个功能强大的前端框架

步骤：

  1. 在< title >标签后加载Bootstrap的层叠样式表：< link href="< ?php echo base_url('Static/bootstrap/css/bootstrap.min.css')?>" rel='stylesheet'>
  2. 在< body >标签后加载jQuery库：<script src="<?php echo base_url('Public/js/jquery.js')?>"></script>
  3. 然后再加载Bootstrap的核心js库：< script src="<?php echo base_url('Static/bootstrap/js/bootstrap.min.js')?>"></script >

注意：

加载Bootstrap.min.js之前必须加载jQuery库，否则无法正常使用

响应式表格

将任何 .table 元素包裹在 .table-responsive 元素内，即可创建响应式表格，其会在小屏幕设备上（小于768px）水平滚动。当屏幕大于 768px 宽度时，水平滚动条消失。

基本实例
